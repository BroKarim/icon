# iicons

**Icon Explore with Infinite Canvas**

</div>

A modern icon explorer built with Vue 3 and Vite. Browse and search 200,000+ icons from Iconify with an intuitive infinite canvas interface.

> **Fork of** [antfu-collective/icones](https://github.com/antfu-collective/icones)  
> **Icons powered by** [Iconify](https://github.com/iconify/iconify)

## ✨ Features

- [x] **Infinite Canvas** - Pan and drag to explore icons in a 2D space
- [x] **Global Search** - Search across all Iconify collections with fuzzy matching
- [x] **Icon Resize** - Adjust icon size with slider control (32px - 128px)
- [x] **Color Picker** - Change icon color with preset color palette
- [x] **Icon Detail** - View icon details with copy/download options (SVG, PNG, Vue, React, etc.)
- [x] **Responsive Design** - Works on desktop and mobile
- [x] **PWA Support** - Install as progressive web app for offline access
- [ ] **Collection Browse** - Browse icons by collection/category
- [ ] **Collection Detail** - View all icons in a specific collection
- [ ] **Favorites/Bookmarks** - Save favorite icons for quick access
- [ ] **Dark Mode** - Toggle between light and dark themes
- [ ] **Keyboard Shortcuts** - Quick actions via keyboard
- [ ] **Icon Suggestions** - Get suggestions for similar icons

## 🚀 Usage

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```
